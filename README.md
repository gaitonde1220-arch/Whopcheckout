# WhopCheckout

Whop payment gateway for WooCommerce — sales funnel + ship-ready plugin.

## Folders

| Folder | Purpose |
|--------|---------|
| **`funnel-site/`** | WhopCheckout.com marketing site (deploy this) |
| **`Ready whop for wp/`** | Plugin v5.0.1 source, ZIP build, seller & buyer docs |
| `whop-gateway-wc/` | Full Connect/OAuth edition (optional) |

## Quick start

**Preview funnel:**
```bash
cd funnel-site && python3 -m http.server 8080
```

**Deploy funnel:** Upload `funnel-site/` to Netlify, Cloudflare Pages, or any static host.

**Sell plugin:** See `Ready whop for wp/INSTRUCTIONS-SELLER-LEMONSQUEEZY.md`
