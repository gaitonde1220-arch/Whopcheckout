Whop Checkout for WooCommerce — setup wizard and webhook-verified payments. No OAuth server required.

== Description ==

**Whop Checkout** connects your WooCommerce store to Whop with a guided 5-step setup wizard.

= Highlights =

* **Setup Wizard** — API key, webhook, sandbox test
* **Connection Health** dashboard
* **Webhook-verified** — orders marked paid only after signed Whop webhooks
* **Sandbox mode** — test before going live
* **HPOS compatible**
* **No Connect server** — works on any host

== Installation ==

1. Upload and activate the plugin.
2. Go to **WooCommerce → Whop Checkout → Setup Wizard**.
3. Complete all 5 steps including a sandbox test order.
4. Disable sandbox for live sales.
