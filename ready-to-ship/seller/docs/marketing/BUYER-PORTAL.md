## Quick start (2 paths)

### Path A — Connect your Whop (recommended, ~2 minutes)

1. WordPress → Plugins → Upload → Activate
2. WooCommerce → **Whop Checkout** (left menu)
3. Enter your **license key**
4. Click **Connect your Whop**
5. Log in to Whop → Approve
6. Run **sandbox test order** → disable sandbox → go live

### Path B — Manual setup (~15 minutes)

1. Install and activate the plugin
2. Open **Whop Checkout → Setup Wizard**
3. Follow all 5 steps (API key, webhook, test)

Full manual guide: [SETUP.md](../SETUP.md)

## FAQ

**Do I need my own Whop account?**  
Yes. Payments go to your Whop company, not the plugin seller.

**Is HTTPS required?**  
Yes, for webhooks on production stores.

**Classic checkout only?**  
Yes for v4.0. Block checkout coming in a future update.

**Can I use on multiple sites?**  
Starter: 1 site. Agency: unlimited (your policy).
