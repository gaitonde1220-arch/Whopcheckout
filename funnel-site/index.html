FAQ

Whop WooCommerce plugin — frequently asked questions

Everything you need to know about Whop Checkout for WordPress and WooCommerce.

What is Whop Checkout for WooCommerce?

Whop Checkout is a WordPress plugin that adds Whop as a payment gateway on WooCommerce. It automatically creates a Whop checkout session from whatever is in the customer's cart — no separate Whop plan ID per product.

Do I need a Whop plan ID for every WooCommerce product?

No. Whop Checkout generates checkout sessions dynamically from your cart total. You do not need to create individual Whop plan IDs for each product, price, or variant.

What are the requirements for Whop Checkout?

You need WordPress 6 or newer, WooCommerce 7 or newer, HTTPS on your live site, and your own Whop business account with API keys. Classic WooCommerce checkout is required.

Does Whop Checkout support sandbox testing?

Yes. The plugin includes a sandbox mode and setup wizard so you can test orders end-to-end before going live with real payments.

How much does Whop Checkout cost?

Whop Checkout is a one-time payment of $170 for lifetime plugin access, unlimited stores, setup wizard, webhook registration, and email support.

Where do customer payments go?

Payments go to your Whop business account — not ours. You connect your own Whop API keys during the 5-minute setup wizard.
